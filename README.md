## Features

* Supporting editors & authors by providing
    * IP restrictions of pages and content elements via:
        * single IP e.g. 192.168.0.1
        * IP range e.g. 192.168.0.1-192.168.0.255
        * IP subnet e.g. 192.168.0.1/24
